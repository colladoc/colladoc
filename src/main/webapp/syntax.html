[q3] Text "Foo" in comments. // Foo Comments that contain text "Foo".
[q4] Classes\objects\traits\packages named "Foo". class Foo; object Foo; trait Foo; package Foo Entities named "Foo" exactly.
[q5] Classes\objects\traits\packages whose names end with "Foo" class _Foo; object _Foo trait _Foo; package _Foo Entities with names ending with "Foo". e.g. Foo, MyFoo
[q6] Classes that extend class "BaseFoo" extends BaseFoo All classes that derive from "BaseFoo". e.g. DerivedFoo
[q7] Classes that implement trait "FooTrait" with FooTrait All classes that implement the trait "FooTrait".
[q8] Methods named "bar" def b def _ : Int All methods that return Int
[q10] Methods that have two parameters, an Int and a String, and return Int def _(Int, String): Int All methods that take an Int and a String and return Int. e.g. bar(i: Int, s: String) : Int This will also return equivalent curried functions e.g. bar(Int)(String)
[q11] Method that takes an Int parameter, followed by zero or more parameters def _(Int, *) All methods that take an Int and zero or more parameters
[q12] Values or variables called "name" val name:String var name:String All member values or variables that are named "name" and are of type String. The results are ordered by val / var depending on the query.
[q13] Search using logical AND. e.g. search for classes named "Foo" and classes that contain a method "bar". class Foo AND def bar class Foo && def bar All classes named Foo that contain a method bar
[q14] Search using logical OR. e.g. search for methods named "add" or methods named "sub" def add OR def sub def add || def sub All methods named "add" or methods named "sub".
[q15] Search using the NOT operator. e.g. search for classes that end with text "Foo", but exclude classes named "BaseFoo" from the results class _Foo not class BaseFoo class _Foo !class BaseFoo All classes that end with "Foo" but not class "BaseFoo"
[q16] Search using query grouping. e.g. search for classes named "Bar" or for classes with names ending in "Foo" but not "BaseFoo" class Bar OR (class _Foo NOT class BaseFoo) All classes named Bar in addition to classes that end with "Foo" but not class "BaseFoo"
[q17] Search for method that takes a function as a parameter. def _(Int => String) All methods that take one parameter which is a function with signature Int => String
[q18] Search for all methods with two parameters, an Int and a String, and return Int Int => String => Int All methods that take an Int and a String and return Int. e.g. bar(i: Int, s: String) : Int
[q19] Search for a generic class with one parameter class _[A] All classes with one generic parameter
[q20] Search for a generic method called "foo" with two generic parameters. def foo[A,B](A, B, Int) Returns all methods called foo with two generic parameters and the respective argument types.
